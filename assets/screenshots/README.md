# Screenshots

This directory contains application screenshots for the README.

## Required Screenshots

- `issue_list_screen.png` - Main issue list screen showing the application interface
- `server_config_screen.png` - Server configuration screen for initial setup

## Adding Screenshots

To add a screenshot:

1. Take a screenshot of the application running on a device or emulator
2. Save it with the appropriate filename in this directory
3. Ensure the screenshot is clear and shows the key features
4. Recommended resolution: 1080x1920 (portrait) or 1920x1080 (landscape)

## Screenshot Descriptions

### Issue List Screen
Shows the main interface with:
- Issue cards with status indicators
- Search bar at the top
- Filter and refresh buttons
- Floating action button for creating new issues

### Server Configuration Screen
Shows the initial setup interface with:
- Application logo
- Welcome message
- Server URL input field
